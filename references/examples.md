# 具体例 — 中学教材サイト

## 例1: 歴史「鎌倉幕府の成立」（中1・基本）

### サイトマップ
```
index.html          … トップページ（単元一覧）
kamakura-intro.html … 導入：源平の争いから鎌倉へ
kamakura-main.html  … 要点＋例題：鎌倉幕府のしくみ
kamakura-quiz.html  … 練習問題＋解説
```

### 導入テキスト例
> 平安時代の終わりごろ、武士の力がどんどん強くなっていきました。
> 源頼朝（みなもとのよりとも）は、弟の義経（よしつね）たちとともに
> 平氏をたおし、1185年に全国に守護・地頭を置きました。
> そして1192年、征夷大将軍に任命され、鎌倉に幕府を開きました。

### 要点まとめ例
- **守護**（しゅご）：国ごとに置かれ、軍事・警察をになう
- **地頭**（じとう）：荘園・公領ごとに置かれ、年貢の取り立てや治安を守る
- **御恩と奉公**：将軍が御家人に土地を与え（御恩）、御家人は将軍のために戦う（奉公）

### 例題
> 鎌倉幕府で、将軍と御家人の関係を表す言葉を答えなさい。
>
> **解説**: 将軍は御家人に土地（御恩）を与え、御家人は戦のときに将軍のために戦います（奉公）。この関係を「御恩と奉公」といいます。

### 練習問題（選択式）
> **問1**: 源頼朝が征夷大将軍に任命されたのは何年ですか？
> - ア. 1185年
> - **イ. 1192年** ← 正答
> - ウ. 1221年
> - エ. 1232年

### 練習問題（穴埋め式）
> **問2**: 国ごとに置かれて軍事・警察をになった役職を（　　）という。
> **答え**: 守護

## 例2: 歴史「室町時代の文化」（中1・標準）

### サイトマップ
```
index.html              … トップページ
muromachi-intro.html    … 導入：南北朝の統一と室町幕府
muromachi-culture.html  … 要点＋例題：北山文化と東山文化
muromachi-quiz.html     … 練習問題＋解説
muromachi-summary.html  … まとめ：鎌倉〜室町の比較
```

### 導入テキスト例
> 足利義満（あしかが よしみつ）は南北朝を統一し、室町幕府の力を強めました。
> 義満のころの文化を**北山文化**（きたやまぶんか）といいます。
> 代表的な建物が「金閣」です。
>
> その後、足利義政（よしまさ）のころには、より落ち着いた雰囲気の
> **東山文化**（ひがしやまぶんか）が生まれました。代表的な建物が「銀閣」です。

### 要点まとめ例
- **北山文化**: 華やか・公家と武家の融合。金閣、能（観阿弥・世阿弥）
- **東山文化**: わび・さびの精神。銀閣、書院造、水墨画（雪舟）、茶の湯・生け花
- **比較ポイント**: 金閣＝金箔で華やか ↔ 銀閣＝質素で落ち着き

### 練習問題（選択式）
> **問1**: 東山文化の代表的な画家で、水墨画を大成した人物はだれですか？
> - ア. 観阿弥
> - イ. 世阿弥
> - **ウ. 雪舟** ← 正答
> - エ. 足利義政

### 練習問題（穴埋め式）
> **問2**: 足利義満のころの文化を（　　）文化という。
> **答え**: 北山

### 解説テキスト例
> 問1: 雪舟（せっしゅう）は室町時代の画僧で、中国に渡って水墨画を学びました。
> 代表作に「天橋立図」などがあります。観阿弥・世阿弥は能を大成した親子です。
>
> 問2: 足利義満は京都の北山に金閣を建てたため、このころの文化を「北山文化」と呼びます。
